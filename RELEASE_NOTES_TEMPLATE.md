# Release Notes Template

Use this template for `--changelog` text when publishing a new version.

## Patch Release

For `0.1.x` style updates:

`Refined public skill routing, documentation, and release maintenance flow.`

## Minor Release

For new public behavior or significant workflow changes:

`Expanded hub routing and upgraded public skill contracts for PPT, social ops, and promo video workflows.`

## Long Form Draft

If you want to draft first and then compress into one line:

- what changed in routing
- what changed in public skill boundaries
- what changed in references or workflow contracts
- whether any public package was added or removed
