# Style Presets

## Consulting

适合：
- investor
- sales
- board update
- project update

特征：
- 结论式标题
- 清晰网格
- 证据卡片和图表面板
- restrained 配色
- 信息密度可控，但不杂乱

常见模板家族：
- `board_evidence`
- `proposal_consulting`
- `research_editorial`

推荐配色方向：
- `ivory_blue`
- `slate_blue`
- `ink_blue`

默认版式：
- `headline_support`
- `evidence_panel`
- `process_map`
- `decision_close`

## Product Launch

适合：
- keynote
- product reveal
- demo talk

特征：
- 大标题
- 强留白
- hero visual
- 深色背景或高对比背景
- 更少元素，更强节奏

常见模板家族：
- `hero_dark_launch`
- `demo_reveal`

推荐配色方向：
- `midnight_cyan`
- `obsidian_lime`

默认版式：
- `hero_reveal`
- `headline_support`
- `decision_close`

## Brand Marketing

适合：
- campaign deck
- external storytelling
- brand presentation

特征：
- mood 更强
- 图像主导
- 配色更有态度
- 文案更短

常见模板家族：
- `editorial_campaign`
- `image_magazine`

推荐配色方向：
- `terracotta_editorial`
- `plum_editorial`

## Minimal Data

适合：
- training
- internal operating review
- clear documentation-style decks

特征：
- 低噪音
- 直接标签
- 简单图表
- 强调扫描效率

常见模板家族：
- `training_manual`
- `clean_report`

推荐配色方向：
- `teal_neutral`
- `stone_green`

## Default Rule

先按场景选 style，再按 style 选版式。不要先追求“好看”，再勉强把内容塞进去。

再进一步：
- 先选 `style`
- 再选 `template family`
- 再选 `palette recipe`
- 最后才做页面级排版和配图

模板家族负责“像什么场景里的成熟 deck”，配色方案负责“整体气质和辨识度”，两者不要混为一谈。
