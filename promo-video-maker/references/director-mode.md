# Director Mode

Director mode is project mode with staged control.

The user is the director.
The system is the production crew.

## What director mode should do

- create or resume a project
- quote the next phase
- run the next phase
- persist results
- show what is ready, blocked, or failed
- hand the user back to the web workspace when useful

## What it should not do

- hide all intermediate stages in a black box
- create duplicate projects without checking
- pretend rough generated assets are a finished episode

## Suggested phase targets

- `story-bible`
- `character-pack`
- `scene-pack`
- `storyboard`
- `audio`
- `video`
- `full-episode`
