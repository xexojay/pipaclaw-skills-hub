# Worker Playbooks

## `Showrunner`

Use when:
- the brief is ambiguous
- mode must be chosen
- production path must be selected
- a phase is blocked and rerouting is needed

Must decide:
- `one-shot` or `project`
- which phase is next
- whether quote should be single-stage or full-project

## `Head Writer`

Use for:
- story premise
- synopsis
- script
- dialogue polish
- episode planning
- novel-to-script style tasks

## `Character Director`

Use for:
- character cards
- visual traits
- identity anchors
- generation prompts for character looks

## `Scene Director`

Use for:
- setting design
- environment packs
- scene references
- location continuity

## `Storyboard Director`

Use for:
- beat sheets
- shot lists
- camera intent
- storyboard prompts

## `Voice Director`

Use for:
- narration strategy
- speaker mapping
- tone and emotion per line
- dialogue audio planning

## `Music Director`

Use for:
- BGM brief
- cue map
- emotional curve
- silence / impact moments

## `Edit Director`

Use for:
- assembly brief
- pacing plan
- transitions
- subtitles and on-screen text guidance

## `Pipeline Operator`

Use for:
- quote calls
- create/list/select project
- execute generation APIs
- poll tasks
- persist results
- produce workspace access
