# Promo Video Maker

Turn a product or campaign brief into a production-ready video plan. Start from audience, placement, duration, and CTA, then develop the concept, script, shots, and edit guidance.

## Intake

Collect or infer:
- objective, audience, and CTA
- publishing surface and aspect ratio
- target duration
- brand voice and proof points
- available footage, product UI, testimonials, or b-roll
- legal, claim, and music constraints

If the user wants a video without assets, create a plan that clearly separates required assets from optional assets.

## Workflow

1. Choose the format.
Read `references/video-formats.md` and select a structure that fits the placement, duration, and conversion goal.

2. Define the concept.
Write the core promise, emotional angle, and visual logic. Keep one dominant message per video.

3. Draft the script.
Write by beat, not by paragraph. Show hook, proof, payoff, and CTA timing.

4. Build the shot list.
Turn the script into concrete shots, visuals, motion notes, and asset requirements.

5. Prepare the handoff.
Package edit notes, caption suggestions, on-screen text, and versioning guidance for multiple placements if needed.

## Output Contract

Return a production pack with:
- creative brief summary
- concept statement
- timed script or beat sheet
- shot list or storyboard table
- edit guidance
- CTA and versioning suggestions

Prefer structured tables and timestamps when possible.

## Guardrails

Do not promise visuals, claims, or testimonials that the user cannot legally or practically use.

Keep the first seconds strong. If the opening does not create curiosity or immediate value, rewrite it.

If placement-specific best practices matter and may have changed recently, verify them before giving final tactical advice.

Separate the master narrative from placement-specific versions. Do not force one script to serve every format unchanged.
